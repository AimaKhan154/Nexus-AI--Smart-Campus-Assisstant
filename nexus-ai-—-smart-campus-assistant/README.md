# 🌟 Nexus AI – Smart Campus Ecosystem

Nexus AI is a futuristic, smart campus ecosystem designed for Sir Syed University. It uses a **Quantum/Cyberpunk design language** to provide students with a high-tech interface for academic and social life.

---

## 🚀 Core Features

### 🤖 AI Smart Assistant (Chat)
A central hub powered by **AI** that:
- Answers student queries  
- Provides campus information  
- Helps with academic tasks  

### 🧠 Neural Study Helper
An intelligent study companion that:
- Summarizes notes  
- Creates study schedules  
- Breaks down complex topics  

### 📊 Dynamic Dashboard
A real-time control center showing:
- Student stats  
- Recent activities  
- System synchronization status  

### 🗺️ Campus Navigator
An interactive map system to locate:
- Departments  
- Labs  
- Campus facilities  

### 👥 Peer Connect
A social networking portal to:
- Find study groups  
- Connect with students from different departments  

### 👨‍🏫 Faculty Directory
A smart search system for:
- Contacting professors  
- Finding university staff  

### 📢 Smart Announcements
A real-time feed for:
- University events  
- Academic notices  

### 🔐 Biometric Authentication
A secure login/signup system with:
- Futuristic "Neural Profile" registration theme  

---

## 🛠️ Technical Stack

### 🎨 Frontend (User Interface)
- **React 19** – Core framework for a fast SPA  
- **TypeScript** – Strongly typed, reliable codebase  
- **Tailwind CSS 4** – Modern UI styling with glassmorphism  
- **Framer Motion / Motion** – Smooth animations and transitions  
- **Lucide React** – Clean, modern icon set  
- **React Router** – Seamless navigation  

### ⚙️ Backend (Engine)
- **Node.js & Express** – API and server logic  
- **JWT (JSON Web Tokens)** – Secure authentication  
- **Bcrypt.js** – Password hashing for security  
- **Tsx & Esbuild** – Fast TypeScript runtime and build tools  

### 🗄️ Database 
- **MongoDB (Mongoose)** – Primary database
